# Architecture Brief

## Purpose

Initiate an Architecture Review Board (ARB) session by defining the problem, objective and review scope.

## Scope

One architectural topic, feature domain or documentation deliverable under review.

## Required Inputs

| Input | Source |
|-------|--------|
| Review topic title | ARB session initiator |
| Related official docs | `docs/` reference list |
| Stakeholders | Role assignments |
| Target working folder | `architecture-board/30/` – `35/` |
| Session date | ARB coordinator |

## Output Format

Completed brief copied to the target working folder as `ArchitectureBrief.md`.
