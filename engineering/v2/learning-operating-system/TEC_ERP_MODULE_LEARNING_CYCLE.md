# TEC.ERP Module Learning Cycle

> **Status:** CONTRACT — NOT IMPLEMENTED · Wave 1 Learning Architecture · Do not treat as shipped UX.

**Authority:** TEC.ERP Learning Operating System Master Contract · Owner review required before Wave 2.

---

Every V2 module SHALL follow this official cycle in order:

1. **DÉCOUVRIR**  
2. **APPRENDRE**  
3. **OBSERVER**  
4. **PRATIQUER**  
5. **ANALYSER**  
6. **RÉFLÉCHIR AVEC L’IA**  
7. **VALIDER**  
8. **DÉBRIEFER**

## Phase definitions

### 1. DÉCOUVRIR
Visual module introduction; why it matters; departments; process map; business context; expected outcome. First viewport MUST NOT be a wall of text.

### 2. APPRENDRE
Microlearning; concepts; glossary; examples; common errors; quantitative example; knowledge check.

### 3. OBSERVER
Professor demonstration (live or Teaching Deck); ERP screen; document; transaction; KPI; decision; downstream effect.

### 4. PRATIQUER
Exactly three missions with guided → autonomous progression; evidence production; recovery paths.

### 5. ANALYSER
BI Studio; KPI reading; filters; trends; causes; decision. Charts without interpretation are non-compliant.

### 6. RÉFLÉCHIR AVEC L’IA
AI Decision Workspace; evidence-backed reasoning; Socratic guidance; learner synthesis required.

### 7. VALIDER
Mission evidence; assessment; competency validation; remediation; Professor approval where required.

### 8. DÉBRIEFER
Learning synthesis; process integration; previous/next module connection; reflection; next-step readiness.

## Experience mode overlay (Wave 1A)

| Cycle phase | Typical official mode |
|-------------|----------------------|
| DÉCOUVRIR / APPRENDRE / OBSERVER | Exploration or Guided Practice |
| PRATIQUER Mission 1–2 | Guided Practice → Simulation |
| PRATIQUER Mission 3 | Simulation |
| ANALYSER / RÉFLÉCHIR | Simulation support (Visible AI) |
| VALIDER | Evaluation where assessed |
| DÉBRIEFER | Facilitation package AFTER + reflection/reapplication |

Mode declaration remains mandatory (`TEC_ERP_EXPLORATION_GUIDED_SIMULATION_EVALUATION_CONTRACT.md`).

## Non-compliance

Skipping ANALYSER or RÉFLÉCHIR, or collapsing three missions into identical tasks, violates this contract. Skipping debrief or reapplication after major simulation violates Wave 1A contracts.
