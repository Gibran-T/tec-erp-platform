Mastery SHALL NOT be reduced to a single score.

## Mastery states

`NOT_OBSERVED` · `EMERGING` · `DEVELOPING` · `DEMONSTRATED` · `CONSISTENT` · `NEEDS_REINFORCEMENT`

## Evidence sources

missions · assessment · recovery · reflection · documents · transactions · BI · AI-assisted reasoning · Capstone · Professor observation

## Competency record fields

| Field | Rule |
|-------|------|
| title | Human label FR/EN |
| description | Observable capability |
| module sources | Contributing modules |
| mission sources | Contributing missions |
| minimum evidence | Count + types |
| evidence diversity | ≥2 categories before DEMONSTRATED |
| recency | Prefer recent successful evidence |
| consistency | Repeated success → CONSISTENT |
| recovery effect | Successful recovery can raise from NEEDS_REINFORCEMENT |
| Professor judgment | May confirm/override with rationale |
| confidence | Low/Med/High based on diversity + recency |
| limitations | Explicit unknowns |
