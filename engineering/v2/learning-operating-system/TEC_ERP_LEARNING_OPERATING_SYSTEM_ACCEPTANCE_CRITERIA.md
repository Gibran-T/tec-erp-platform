## Wave 1 baseline criteria (preserved)

Future waves MUST satisfy:

1. Learner understands current stage within five seconds.  
2. Every module has a visible process map.  
3. Every module has a Teaching Deck.  
4. Every mission has a distinct pedagogical role (1/2/3).  
5. BI contains real visualizations with interpretation path.  
6. AI uses evidence and a large workspace; learner synthesis required.  
7. Historical runs expose no write action (Consulter only).  
8. V2 M8=HCM, M9=Governance, M10=BI/KPI/AI correct.  
9. Capstone outside the 30; no M11.  
10. Professor can teach from the platform (Deck + demo + BI).  
11. Learner can review after class (learner-safe deck).  
12. No wall-of-text first viewport.  
13. Visual assets load; no broken image strategy.  
14. FR/EN complete on new surfaces.  
15. Light / Dark / projector modes tested.  
16. V1 James Run 1 remains immutable across all waves.  
17. No secrets in repo.  
18. Gold V2 requires HCM where applicable; V1 history preserved.

## Wave 1A Research Addendum criteria (testable)

19. Learner understands active role and mandate within five seconds.  
20. Every important action creates visible consequence.  
21. Every important action affects at least one governed dimension (ops/finance/service/workforce/governance/KPI/risk).  
22. Stakeholder responses reference current enterprise state.  
23. Stakeholder responses respect authority.  
24. Stakeholder responses do not contradict documents.  
25. Visible AI and ambient AI are distinguishable.  
26. AI-generated text never mutates official state by itself.  
27. Historical runs receive no new stakeholder events.  
28. Exploration produces no official mutation.  
29. Guided Practice separates help from evidence.  
30. Simulation records evidence and consequences.  
31. Evaluation restricts assistance.  
32. Every simulation includes debrief.  
33. Every reflection includes reapplication.  
34. Novice guidance is stronger than advanced guidance.  
35. No mandate begins with a wall of text.  
36. Every KPI movement has traceable cause.  
37. Every forecast is labeled.  
38. Professor can control or suppress scenario events.  
39. AI-unavailable fallback preserves core learning.  
40. No important learner action disappears silently.
